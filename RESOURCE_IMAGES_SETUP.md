# Resource Images Setup

## Where to Put Images

Place all resource images in: `/public/images/`

## Image Naming

Images should be named exactly as specified in `lib/game/icons.ts`:

### Metals
- `metal_solvium_gem.png`
- `metal_lunargent_shard_a.png`
- `metal_aethersteel_ingot_a.png`
- `metal_virelith_shard_a.png`
- `metal_noctyrium_orb_a.png`
- `metal_crownlite_shard_a.png`

### Forms
- `resource_cinder_ore_pile_a.png` (Ore)
- `ore_brown_nuggets.png` (Nugget)
- `metal_aethersteel_ingot_a.png` (Bar - reuses aethersteel)
- `resource_mossy_orb_a.png` (Coin)

### Biomes
- `ore_sandstone.png` (Desert)
- `ore_charcoal.png` (Rift)
- `ore_ice_cluster.png` (Glacier)

### Hazards
- `ore_gray_gold_speck.png` (cave_in)
- `resource_aether_ooze_a.png` (gas_pocket)
- `resource_verdant_plate_a.png` (equipment_jam)
- `ore_crimson_lump.png` (misfire)

## Current Status

The icon mapping system is ready. Once you add the images to `/public/images/`, they will automatically be used throughout the game.
